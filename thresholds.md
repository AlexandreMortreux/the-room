# THE ROOM — Day-30 Kill Test

**Locked 2026-07-08.** Pre-registered so day 30 is an honest measurement, not
a moved goalpost. **Day 0 = first public English post.**

## PRIMARY — demand (this IS the go/no-go)

1. Organic subscribers ≥ 200
2. Poll participation ≥ 10% of post views
3. Day-28 post views ≥ 60% of peak post

**Rule:** 3/3 → phase 2. 1–2/3 → one 30-day format iteration, once.
0/3 → shutdown, no regrets.

## SECONDARY — diagnostics (explain the result, never replace it)

- Directional accuracy ≥ 55% over ≥ 20 resolved calls
- Cumulative Brier of at least one agent < 0.25
- Agent records diverge (not both ~50/50)

---

The SECONDARY diagnostics are computed from [`ledger.csv`](ledger.csv);
PRIMARY demand metrics are read from Telegram channel stats. Primary decides;
secondary only explains *why* demand did or didn't show up.
